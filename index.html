Cooltipz.css

Cool tooltips made from pure CSS

Cooltipz.css makes adding tooltips to existing HTML elements easy and simple, without JavaScript.

Current version: v1.3.0

Have a play with Cooltipz.css in CodePen.

Features

  • Easy and simple
  • Pure CSS, no JavaScript
  • Accessible with aria-label attributes
  • Options to use classes or data- attributes
  • Disables animation if user's preference are set to reduced motion
  • Uses Autoprefixer when generating the distributed CSS
  • Import expanded CSS, minified CSS or the the raw Sass/SCSS

Tooltip Direction* - (exactly one direction is required for the tooltip to display)

<button data-cooltipz-dir="top" aria-label="Hello there!">...</button>
<button class="cooltipz--top" aria-label="Hello there!">...</button>
<button data-cooltipz-dir="bottom" aria-label="Hello there!">...</button>
<button class="cooltipz--bottom" aria-label="Hello there!">...</button>
<button data-cooltipz-dir="right" aria-label="Hello there!">...</button>
<button class="cooltipz--right" aria-label="Hello there!">...</button>
<button data-cooltipz-dir="left" aria-label="Hello there!">...</button>
<button class="cooltipz--left" aria-label="Hello there!">...</button>
<button data-cooltipz-dir="top-left" aria-label="Hello there!">...</button>
<button class="cooltipz--top-left" aria-label="Hello there!">...</button>
<button data-cooltipz-dir="top-right" aria-label="Hello there!">...</button>
<button class="cooltipz--top-right" aria-label="Hello there!">...</button>
<button data-cooltipz-dir="bottom-left" aria-label="Hello there!">...</button>
<button class="cooltipz--bottom-left" aria-label="Hello there!">...</button>
<button data-cooltipz-dir="bottom-right" aria-label="Hello there!">...</button>
<button class="cooltipz--bottom-right" aria-label="Hello there!">...</button>

No Animation

Animations can be disabled by apply the data-cooltipz-static attribute. Cooltipz.css also use the prefers-reduced-motion media query, so when the user's preferences indicate reduced motion, Cooltipz.css automatically disables animation.

<button data-cooltipz-dir="top" data-cooltipz-static aria-label="Hello there!">...</button>
<button class="cooltipz--top cooltipz--static" aria-label="Hello there!">...</button>

Always Visible

Add the data-cooltipz-visible attribute or the .cooltipz--visible class to make the tooltip always show, even if the user isn't hovering or focusing it.

Tooltips can be shown programatically by using the setAttribute() JavaScript function.

<button data-cooltipz-dir="top" data-cooltipz-visible aria-label="I'm always visible!">...</button>
<button class="cooltipz--top cooltipz--visible" aria-label="I'm always visible!">...</button>

Customisation

Cooltipz.css exposes 5 CSS variables that can be overriden using your own styles: --cooltipz-bg-color, --cooltipz-text-color, --cooltipz-font-size, --cooltipz-slide and --cooltips-border-radius.

To override every tooltip in your project, place your CSS variables in the :root selector

:root {
  --cooltipz-bg-color: #f00;
  --cooltipz-text-color: #00f;
  --cooltipz-font-size: 1.5rem;
  --cooltipz-slide: 20px;
  --cooltips-border-radius: 1rem;
}

Or override specific tooltips

.custom-cooltip {
  --cooltipz-bg-color: #fff;
  --cooltipz-text-color: #000;
  --cooltipz-font-size: 1.5rem;
  --cooltipz-slide: 50px;
  --cooltips-border-radius: 1rem;
}

Installation

Link to stylesheet

Link to CDN stylesheet in your <head>

<!-- This will get the latest expanded stylesheet -->
<link rel="stylesheet" href="https://raw.githack.com/JDomleo/Cooltipz.css/master/cooltipz.css" />

<!-- This will get the latest minified stylesheet -->
<link rel="stylesheet" href="https://raw.githack.com/JDomleo/Cooltipz.css/master/cooltipz.min.css" />

Link to CDN stylesheet in your CSS

/* This will get the latest expanded stylesheet */
@import url(https://raw.githack.com/JDomleo/Cooltipz.css/master/cooltipz.css);

/* This will get the latest minified stylesheet */
@import url(https://raw.githack.com/JDomleo/Cooltipz.css/master/cooltipz.min.css);

Download it from GitHub and link to it locally

<!-- Be sure to manually update this regularly -->
<link rel="stylesheet" href="path/to/cooltipz.min.css" />

Via npm

Install it via npm

npm install cooltipz-css --save

Import it into your JavaScript

import 'cooltipz-css';

Or import it into your Sass/SCSS

/* Webpack */
@import '~cooltipz-css/src/cooltipz';

/* Non webpack */
@import 'path/to/node_modules/cooltipz-css/src/cooltipz';

Beware

Important: Cooltipz.css uses the ::before and ::after psuedo elements to display the tooltips, therefore you must beware of the following:

  • Cooltipz.css will not work on self-closing HTML elements.
  • If the HTML element already has existing ::before or ::after styling, these could conflict with the Cooltipz.css styles and cause unexpected outputs.
